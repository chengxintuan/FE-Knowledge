# 前言

JS 的知识体系，可以分为：文法、语义、运行时。也就是“用一定的词法和语法，表达一定语义，从而操作运行时”。

从文法和运行时的角度入手，我们就可以了解完整的JavaScript。而语义是文法到运行时之间的桥梁，在讲文法和运行时的同时也会提到，所以不单独讲。

综上，JS部分的重点落在文法和运行时上。
